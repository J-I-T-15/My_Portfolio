Project Overview

The Word Guessing Game, a fun and challenging interactive experience built entirely with HTML, CSS, and JavaScript!

The game presents you with a secret word, concealed by a series of underscores, each representing a letter in the word. Your mission is to deduce the letters and guess the complete word correctly.

You have multiple attempts to guess the word. With each correct guess, the hidden letters are revealed, making it easier to unravel the mystery. The game leverages the power of JavaScript to handle user input, check the correctness of guesses, and update the display accordingly. It employs arrays to store the secret words.

Whether you're playing on a desktop, laptop, or smartphone, the Word Guess Game is fully responsive.
